# E_Commerce_Sales_Project
Performing four types of analysis by doing basic data exploration in Excel transforms a raw product dataset into actionable insights. 
* Basic Data Cleaning was done.
* Inconsistent data in the Customer_ID and Country Column were identified and corrected.
* Blanks in Loyalty_Level were filled with less loyalty_level.
* Inconsistent data in Product_ID were corrected.
* Blanks in the Cost and Average columns were filled with Average by the IF(ISBLANK()) formula.
* Blank Discount % cells filled with average discount value using IF(ISBLANK()) formula.
* Using filter Blanks in Quantity is identified and filled with formula =L2/(G2-(G2*H2)).
* Using filter Blanks in Unit_Price is identified and filled with formula =(L21/(1-H21))/F21.
* Using filter Blanks in Total_Amount is identified and filled with formula =F2*(G2-(G2*H2)).
* New Original amount and Discount Value Columns were added.
* Conditional formatting applied to Payment type and Total Amount.
New sheets were created to find the insights using the Pivot Tables.
Insights were found by analysing the raw dataset. Charts and Slicers were used to create the Interactive Dashboard.
